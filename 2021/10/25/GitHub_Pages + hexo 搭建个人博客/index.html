部署 GitHub pages

在 GitHub 上申请令牌

打开 GitHub 上的 Settings，选择 Developer settings 选项卡，然后选择 Personnal access tokens 选项卡。

点击 Generate new token 按钮，输入密码。

在新建令牌页面中选择所有的权限（其实不用全选，但是笔者懒得挑了）。

点击生成令牌，在接下来的页面中截图或者复制得到的令牌，令牌只会出现一次，一定要记住。
